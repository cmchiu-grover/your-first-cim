操作指南

IE 使用者

- 首頁

首頁直接顯示重要指標 OEE（整體設備效率）

IE User 可以使用本平台所有的查詢功能。

左方功能列表會顯示：OEE、機況圖、機況查詢、在製品查詢與 IE 維護。

其中機況查詢與 IE 維護僅在 IE 使用者時出現

IE 使用者

- OEE 查詢

首頁顯示以站點為主的昨日 OEE 資料

OEE 查詢頁面可查詢其他日期的資料

亦可看更詳細資料：該站點的所有機台 OEE 資料

IE 使用者

- 機況圖

機況圖頁面直接顯示昨日 CPU 站點之機況圖

提供查詢其他天、站點之機況圖

IE 使用者

- 機況查詢

查詢機況詳細資料

直接按下查詢會依照時間排序，由最舊的資料開始顯示所有資料


後續將詳細說明：

EQ 、MFG 使用者在維護功能查詢機況詳細資料

不同在於，其可以填寫、更新 comment 欄位

IE 使用者

- 在製品查詢

查詢機台生產在製品的詳細資料

直接按下查詢會依照時間排序，由最舊的資料開始顯示所有資料


左圖的第一筆資料說明：

CPU 站點之 A001IC 機台，生產 ICC000513 產品 116 個單位

IE 使用者

- 工時維護

IE 使用者可進行工時維護：

1. 使用 csv 檔案維護多筆資料

（下載 example 以照格式填寫）

2. 快速維護 5 筆以內的工時資料（如右圖所示）

IE 使用者

- 工時維護 -維護後通知

IE 使用者工時維護後：

可能會影響作業效率，進而影響 OEE


點擊右上角鈴鐺進入通知頁面

通知頁面會顯示相關影響（或無影響）

IE 使用者

- 工時查詢

查詢產品的詳細工時資料

直接按下查詢會排序顯示所有資料


以右圖的第一筆資料說明：

產品號碼 ICC000513，其產品名為 Markhor

使用 CPU 站點之機台型號 IC01 生產時

IE 工時：每單位所需為 2 分鐘

EQ、MFG 使用者

- EQ、MFG 維護
